Make sure to add the following matlab directory paths to the "Path" environment variabele:
	- <Your Matlab Installation Directory>\runtime\win64, ie. C:\Program Files\MATLAB\R2014a_x64\runtime\win64;
	- <Your Matlab Installation Directory>\bin, ie. C:\Program Files\MATLAB\R2014a_x64\bin;
	- <Your Matlab Installation Directory>bin\win64, ie. C:\Program Files\MATLAB\R2014a_x64\bin\win64

Register the Matlab Server by running this command:
	- matlab /regserver
